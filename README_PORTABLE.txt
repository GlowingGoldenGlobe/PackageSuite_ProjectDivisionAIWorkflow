GlowingGoldenGlobe Portable Division Installer
=============================================

QUICK START:
1. Run GGG_Division_Installer.exe
2. Select packages and installation location
3. Click "Install Division"

CONTENTS:
- installer/          Main installer application
- packages/          All GGG packages organized by type
- dependencies/      Pre-downloaded dependencies for offline install
- templates/         Pre-configured division templates
- docs/             Complete documentation
- tools/            Utility tools for management

REQUIREMENTS:
- Windows 10/11 or Linux
- Python 3.8 or higher
- 10GB free disk space
- Administrator privileges (for some features)

FEATURES:
✓ Offline installation capability
✓ All dependencies included
✓ Automatic configuration
✓ Desktop shortcuts creation
✓ Virtual environment setup
✓ Package selection interface

For detailed instructions, see docs/INSTALLATION_GUIDE.pdf
